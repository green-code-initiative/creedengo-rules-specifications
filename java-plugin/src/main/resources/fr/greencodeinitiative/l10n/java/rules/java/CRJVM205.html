Use lazy fetch type instead of egger.

Noncompliant Code Example

  class FetchTypeLazyCheck {

    @ManyToOne(fetch = FetchType.EAGER)
    @JoinColumn(name="USER_ID")
    private UserLazy user;

   }

Compliant Solution

  class FetchTypeLazyCheck {

        @ManyToOne(fetch = FetchType.LAZY)
        @JoinColumn(name="USER_ID")
        private UserLazy user;

   }

    OR

    class FetchTypeLazyCheck {

        @ManyToOne
        @JoinColumn(name="USER_ID")
        private UserLazy user;

   }
