The time gap will be noticeable within a large number of inclusions. If you are developing a large web application and have hundreds, using an absolute path will save CPU and RAM usage.

Noncompliant Code Example

    include 'menu.php'; // NOK
    include './menu.php'; // NOK
    include '../menu.php'; // NOK
    require 'menu.php'; // NOK

Compliant Solution

    include '/menu.php';
