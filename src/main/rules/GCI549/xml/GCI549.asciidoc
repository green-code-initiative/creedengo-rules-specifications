Hardware acceleration is enabled by default since Android 3.0 (API level 11) for applications targeting API 14+.
While it improves rendering performance, it also increases RAM consumption significantly.

If an application or activity does not require complex 2D rendering, disabling hardware acceleration with `android:hardwareAccelerated="false"` can reduce memory usage and overall power consumption.

== Non compliant Code Example

[source,xml]
----
<application
    android:hardwareAccelerated="true"
    android:label="@string/app_name">
    <activity android:name=".MainActivity"/>
</application>
----

== Compliant Code Example

[source,xml]
----
<application
    android:hardwareAccelerated="false"
    android:label="@string/app_name">
    <activity android:name=".MainActivity"/>
</application>
----
