== Why is this an issue?

Using built-in functions for simple operations can introduce unnecessary overhead.
For basic comparisons, direct operators are more efficient.

== Examples

=== Noncompliant

[source,php,data-diff-id="1",data-diff-type="noncompliant"]
----
$min = min($a, $b); // Noncompliant
----

=== Compliant

[source,php,data-diff-id="2",data-diff-type="compliant"]
----
$min = $a < $b ? $a : $b;
