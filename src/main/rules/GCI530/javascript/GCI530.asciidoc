== Why is this an issue?

Turning on the torch mode programmatically must absolutely be avoided because the flashlight is one of the most energy-intensive component.

== Non compliant Code Example

=== React Native

[source,js,data-diff-id="1",data-diff-type="noncompliant"]
----
import Torch from 'react-native-torch'; // Noncompliant
----

[source,js,data-diff-id="1",data-diff-type="compliant"]
----
import axios from 'axios'; // Compliant
----

=== HTML5 Web API (MediaTrackConstraints)

[source,js,data-diff-id="2",data-diff-type="noncompliant"]
----
// Noncompliant
await track.applyConstraints({ advanced: [{ torch: true }] });
----

[source,js,data-diff-id="2",data-diff-type="compliant"]
----
// Compliant
await track.applyConstraints({ advanced: [{ facingMode: "environment" }] });
----
